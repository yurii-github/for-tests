  - не пустой [100]
  - только цифры, буквы, подчеркивание [101]

  - уникален [103] (не проверяется на клиенте)
 
-- password
  - не пустой [110]
  - не повторяется с username [111]

  - не пустой [120]
  - regex по шаблону [121]
  
-- fullname
  + обрезка пробелов в начале и конце
  + больше одного пробела подряд заменяется на 1 пробел
  - не пустой [130]
  - только буквы и пробел [131]
  - макс 255 символов [132]
  ~ считать первую левую часть именем, все остальное фамилия. доп. запись в name, surname поля

-- sex
  + male/female, по умолчанию male

-- avatar

-- username / password
  - не пустой [200]

  ~ хеширование сессии и пароля
  ~ профиль доступен только зарегистрированному пользователю и только свой профиль
